Ghost Detector - Hata Ayiklama

Bu sayfayi LinkedIn Jobs sayfasinda Console'a (F12) asagidaki komutlari yazarak test edin:

1. Adim: Eklenti yuklu mu?

// Console'a yapiştirin:
console.log("__gd_interval:", window.__gd_interval);
console.log("__gd_observer:", window.__gd_observer);

Eger ikisi de undefined ise content script hic yuklenmemis demektir.

2. Adim: Sayfa DOM'unu kontrol edin

// Console'a yapiştirin:
const selectors = [
  ".job-card-container", "[class*='job-card']", "[data-job-id]",
  "[data-occludable-job-id]", "li[data-occludable-entity-urn]",
  ".jobs-search-results__list-item", ".scaffold-layout__list-item",
  ".jobs-search-results-list__list-item", ".discovery-templates-entity-item",
  "[class*='jobs-home'] li", "[class*='top-job-picks'] li",
  ".artdeco-carousel__item"
];
selectors.forEach(s => {
  const count = document.querySelectorAll(s).length;
  if (count > 0) console.log(`✅ ${s}: ${count} element`);
});
// Fallback
const jobLinks = document.querySelectorAll("li a[href*='/jobs/view']");
console.log(`Fallback (li with job link): ${jobLinks.length}`);
// Tum li'ler
const allLi = document.querySelectorAll("li");
console.log(`Toplam li: ${allLi.length}`);
